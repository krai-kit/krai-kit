# @krai-cdk 
## Tokens

This library contains a set of Angular tokens that are part of the [**@krai-kit**](https://github.com/krai-tech/krai-kit).

## Installation

```bash
npm i @krai-tech/cdk
# or
yarn i @krai-tech/cdk
```
