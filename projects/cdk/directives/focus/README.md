# @krai-cdk
## Focus Directive

Provides a directive to automatically focus.

## Installation

```bash
npm i @krai-tech/cdk
# or
yarn i @krai-tech/cdk
```

## Usage

### Import:

```typescript
import { FocusDirective } from '@krai-tech/cdk/directives/focus';
```

### Use the directive:

```html
<input kriFocus [autofocus]="true">
```
