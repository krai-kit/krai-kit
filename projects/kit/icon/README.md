# @krai-kit
## IconComponent

The `IconComponent` is a reusable component for displaying icons with various customizable properties.

## Installation

```bash
npm i @krai-tech/kit
# or
yarn i @krai-tech/kit
```

## Usage

### Import
```ts
import { IconComponent } from '@krai-tech/kit/icon';
```

### Use the IconComponent:
```html
<kri-icon
  [icon]="iconType"
  [active]="isActive"
  [disabled]="isDisabled"
  [color]="iconColor"
  [size]="iconSize"
  [rotate]="iconRotate">
</kri-icon>
```
